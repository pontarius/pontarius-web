Our work with Pontarius

Our security designs are prototyped, not yet finalized, and have not been vetted by security experts.

We are currently working on the core features of Pontarius; that is, contacts and identity management, and basic secure communication.

The API

Pontarius exposes a simple D-Bus API that allows developers to easily enable Pontarius-based communication to their users. D-Bus is a standardized inter-process communication protocol.

The service

Pontarius is an application running on your machine. It currently uses GnuPG in order to generate identities (cryptographic keys), XMPP for communication, and an Off-the-Record-like protocol for exchanging identities. It aims to allow application developers to establish end-to-end secure communication through something like a virtual TCP socket. The service is written in Haskell, and uses SQLite for storage.
